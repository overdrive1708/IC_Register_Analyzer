# IC_Register_Analyzer

## 【簡単な説明】

ICのレジスタ設定と16進数/10進数/2進数のデータを相互変換するアプリケーションです。  
以下のICに対応しています。  

|IC名|概要|メーカー|
|---|---|---|
|ADF4111|RF PLL周波数シンセサイザ|アナログ・デバイセズ|
|R2A20178NP|8ビット8ch 5V系低消費乗算型D/Aコンバータ(バッファ有り)|ルネサスエレクトロニクス|
